##### 山岡さん

##### マルコスさん
慶應義塾大大学院メディアデザイン研究科 専任講師。
ブラジル・サンパウロ市出身。サンパウロ州立総合大学建築都市計画学部を卒業。1997年に来日し、グラフィック、エディトリアル、広告デザイナーとして活躍。2016年慶應義塾大学大学院メディアデザイン研究科にて博士号を取得。ICTとデジタルメディアを活用したグローバル教育（K-12）について研究。欧米・アジアを対象とした初等・高等教育プログラムの企画、国際機関と連携したプロジェクトの運営を務める。グローバル社会、ダイバーシティ（多様性）、メディアが交わる分野に関心がある。


##### 赤塚さん
都立航空工業高等専門学校在籍中から 4輪レーサーを目指すも断念。蕎麦屋、溶接屋を経てプログラマになる。2003年IPA未踏ユース (スーパークリエイタ)、2006年IPA未踏ソフトウェア創造事業に採択。2011年慶應義塾大学大学院政策・メディア研究科修了。2008年度グッドデザイン賞 (新領域)、2011年慶應義塾大学政策・メディア研究科磯賞受賞。
